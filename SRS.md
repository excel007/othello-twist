# Software Requirements Specification (SRS)
## เกม Othello Twist บนเว็บ
### *"Othello ที่ไม่เหมือนใคร"*

---

## 1. บทนำ

### 1.1 วัตถุประสงค์
เอกสารนี้ระบุความต้องการของซอฟต์แวร์สำหรับพัฒนาเกม **Othello Twist** บนเว็บแอปพลิเคชัน ที่มีกลไกพิเศษสร้างความประหลาดใจและความสนุกที่แตกต่างจาก Othello ทั่วไป

### 1.2 ขอบเขต
- เกม Othello แบบ 1 ผู้เล่น vs AI พร้อมกลไกพิเศษ
- ทำงานบนเว็บเบราว์เซอร์
- รองรับระดับความยาก 3 ระดับ: ง่าย ปานกลาง ยาก

### 1.3 แนวคิดหลัก
> "ทุกสิ่งที่คุณรู้เกี่ยวกับ Othello... อาจไม่เป็นจริงเสมอไป"

---

## 2. ✨ ฟีเจอร์พิเศษ (Surprise Features)

### 2.1 🎲 ระบบ Power-Ups

| Power-Up | ชื่อ | ผลกระทบ | ความถี่ |
|----------|------|---------|--------|
| 💣 | **Bomb** | ระเบิดหมาก 3×3 รอบตำแหน่งที่วาง | สุ่มปรากฏ 10% |
| 🔄 | **Swap** | สลับสีหมากทั้งกระดาน (ดำ↔ขาว) | สุ่มปรากฏ 5% |
| 🛡️ | **Shield** | ป้องกันหมาก 1 ตัวไม่ให้ถูกพลิก 3 ตา | เก็บสะสมได้ |
| ⚡ | **Double Turn** | เดินได้ 2 ครั้งติดต่อกัน | สุ่มปรากฏ 8% |
| 👻 | **Ghost** | วางหมากโดยไม่ต้องขนาบ 1 ครั้ง | สุ่มปรากฏ 7% |
| 🎯 | **Sniper** | พลิกหมากศัตรู 1 ตัวที่ไหนก็ได้ | สุ่มปรากฏ 5% |

### 2.2 🌀 Event แบบสุ่ม (Random Events)

| Event | ชื่อ | ผลกระทบ |
|-------|------|---------|
| 🌊 | **Earthquake** | หมากทั้งหมดเลื่อนไปทิศทางสุ่ม 1 ช่อง |
| 🕳️ | **Black Hole** | ช่องสุ่ม 4 ช่องกลายเป็นหลุมดำ วางหมากไม่ได้ |
| 🌈 | **Rainbow** | หมาก 5 ตัวสุ่มกลายเป็น "หมากทอง" (นับ 2 คะแนน) |
| ⏰ | **Time Warp** | ย้อนเกมกลับไป 3 ตา |
| 🎭 | **Mirror** | กระดานสะท้อนแนวนอน |
| 🌪️ | **Chaos** | หมากสุ่ม 6 ตัวสลับตำแหน่งกัน |

**เงื่อนไข:** Event สุ่มเกิดขึ้นทุก 5-8 ตา โดยมี animation เตือนล่วงหน้า

### 2.3 🏆 ระบบ Combo

| Combo | เงื่อนไข | รางวัล |
|-------|----------|--------|
| **Triple Flip** | พลิกหมาก 8+ ตัวในตาเดียว | ได้ Power-Up สุ่ม 1 ชิ้น |
| **Corner Master** | ยึดมุมได้ | หมากที่มุมกลายเป็นหมากทอง (2 คะแนน) |

### 2.4 🎮 โหมดพิเศษ

#### 2.4.1 โหมด Classic
- เล่นแบบ Othello ดั้งเดิม ไม่มี Power-Up หรือ Event

#### 2.4.2 โหมด Chaos (ค่าเริ่มต้น)
- มี Power-Up และ Random Event เต็มรูปแบบ

#### 2.4.3 โหมด Strategy
- เลือก Power-Up เริ่มต้นได้ 3 ชิ้น (Bomb, Shield, Double)
- ไม่มี Random Event
- เน้นวางแผน

#### 2.4.4 โหมด Speedrun
- Power-Up ปรากฏบ่อยขึ้น

### 2.5 🎨 หมากพิเศษ (Special Discs)

| หมาก | รูปลักษณ์ | คุณสมบัติ |
|------|----------|----------|
| 🥇 **หมากทอง** | สีทอง | นับ 2 คะแนน |

## 3. ความต้องการเชิงฟังก์ชัน

### 3.1 การเริ่มเกม (FR-001)
| รายการ | รายละเอียด |
|--------|------------|
| คำอธิบาย | ผู้ใช้เลือกโหมด ระดับความยาก และเริ่มเกม |
| Input | เลือกโหมด, เลือกระดับความยาก, กดเริ่ม |
| Output | กระดานเกมพร้อมเล่น |

### 3.2 การเลือกระดับความยาก (FR-002)
| ระดับ | พฤติกรรม AI |
|-------|-------------|
| ง่าย | สุ่มเลือก move |
| ปานกลาง | Greedy + ให้ความสำคัญกับมุม |
| ยาก | Minimax + Alpha-Beta Pruning |

### 3.3 การวางหมาก (FR-003)
| รายการ | รายละเอียด |
|--------|------------|
| คำอธิบาย | วางหมากตามกติกา หรือใช้ Power-Up |
| Process | ตรวจสอบ valid move → ตรวจสอบ Power-Up → วางหมาก → trigger effect |
| Animation | แสดง animation พลิกหมาก + effect พิเศษ |

### 3.4 ระบบ Power-Up (FR-004)
| รายการ | รายละเอียด |
|--------|------------|
| การได้รับ | สุ่มปรากฏบนกระดาน เก็บโดยวางหมากทับ |
| การเก็บ | เก็บได้สูงสุด 3 ชิ้น |
| การใช้ | คลิกเลือก Power-Up ก่อนวางหมาก |
| แสดงผล | Icon ที่ด้านล่างพร้อม tooltip อธิบาย |

### 3.5 ระบบ Random Event (FR-005)
| รายการ | รายละเอียด |
|--------|------------|
| Trigger | ทุก 5-8 ตา (สุ่ม) |
| การแจ้งเตือน | แสดง popup + animation ก่อนเกิด |
| หลังเกิด | อัปเดตกระดานทันที |

### 3.6 ระบบ Pass Turn (FR-006)
| รายการ | รายละเอียด |
|--------|------------|
| เงื่อนไข | เมื่อผู้เล่นไม่มี valid move |
| แสดงผล | Popup แจ้ง "PASS!" พร้อมบอกว่าใครถูกข้าม |
| พฤติกรรม | ข้ามตาอัตโนมัติ ให้ฝ่ายตรงข้ามเล่นต่อ |

### 3.7 การนับคะแนน (FR-007)
| ประเภทหมาก | คะแนน |
|------------|--------|
| หมากปกติ | 1 คะแนน |
| หมากทอง 🥇 | 2 คะแนน |

### 3.8 การจบเกม (FR-008)
| รายการ | รายละเอียด |
|--------|------------|
| เงื่อนไข | กระดานเต็ม / ทั้งสองฝ่ายไม่มี valid move |
| แสดงผล | สรุปคะแนน + สถิติ (Combo, Power-Up, Event) |

### 3.9 ระบบช่วยเหลือ (FR-009)
| รายการ | รายละเอียด |
|--------|------------|
| ปุ่มช่วยเหลือ | ปุ่ม ❓ มุมขวาล่าง |
| เนื้อหา | คำอธิบาย Power-Ups, Events, หมากพิเศษ |
| Toast | แสดงคำอธิบายเมื่อเจอสิ่งใหม่ครั้งแรก |

## 4. ความต้องการเชิงไม่ใช่ฟังก์ชัน

### 4.1 ประสิทธิภาพ (NFR-001)
- Animation ลื่นไหล
- AI ตอบสนองภายใน 2 วินาที

- Responsive design รองรับ Mobile และ Desktop
- Touch-friendly สำหรับหน้าจอสัมผัส

### 4.3 ความเข้ากันได้ (NFR-003)
- รองรับ Chrome, Firefox, Safari, Edge
- รองรับหน้าจอขนาด 320px ขึ้นไป

### 4.4 เสียง (NFR-004)

|-------|-------|
| วางหมาก | "Pop" |
| พลิกหมาก | "Flip" |
| ได้ Power-Up | "Power up!" |
| Random Event | Warning |
| Combo | Celebration |
| Pass Turn | Notification |
